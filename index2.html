Questions:-3: What is the event delegation in the DOM?

Event delegation is a way to use one event listener instead of using multiple time. It makes the code easy to write and easy to read.

Questions:-4: What is the event bubbling in the DOM?

It is process where an event begins from the targeted element and reaches up to the parent elements. Thus keep rocking until it reaches the document.
